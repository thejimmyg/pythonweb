1.2 web.auth -- Easy to use authorisation, authentication and user management system

The web.auth module provides methods for allowing multiple users in different groups, multiple access levels to multiple applications with multiple roles and activity status using a single login. It offers a poweful, flexible and simple way to restrict or manage access to different parts of your code and is suitable for enterprise use.
