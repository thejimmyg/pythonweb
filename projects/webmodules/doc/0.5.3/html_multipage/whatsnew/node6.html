1.4 0.5.0

Version 0.5 should be considered the first release of the development branch of the web modules. A lot of changes have been made and more are planned. It is likely that with these changes some new bugs may have been introduced and so a series of 0.5 releases are planned as any problems become apparant.

This version brings with it a number of important changes:

  • Now licenced under the LGPL after discussion on the Web Modules mailing list.
  • Full WSGI support.
  • New database layer which includes a pure Python reference implementation.
  • New coding philosophy with emphasis on one class only performing one task and no over-simplification. Most modules re-written to allow them to be easily extended with new drivers/handlers.
  • Support for the idea of environments.
  • Updated documentation in various formats.
  • Many bug fixes.
