1.5.1.1 Requirements

To use web.database.object you need Python 2.2 or above and the Web Modules of which web.database.object is a part and an SQL database supported by the web.database module and its associated Python driver. If you use MuSQL, a MySQL database is needed, alternatively use an ODBC database such as MS Access. The web.database driver is included with the Python Web Modules but you will need to download and install the ODBC driver from the http://www.eGenix.com site yourself as it comes with a non open source licence.
