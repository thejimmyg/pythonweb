1.7 web.form -- Construction of persistant forms/wizards for HTML interfaces

The web.form module a series of classes and functions for generating and managing persistant HTML forms. As well as basic fields such as input or select fields, the module provides fields for dates, email addresses, URLs and more. It also supports fields which return Python types, for example the Integer Select field or the Date field.

The web.form module also provides a mechanism for automatically handling invalid data and requesting more information from the user.
