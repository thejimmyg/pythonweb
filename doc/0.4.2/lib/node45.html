1.4.9.3 ODBC

This implementation has only been tested with MS Access XP as an ODBC data source. On Windows XP you can set up a data source by using the Data Sources (ODBC) shortcut in Administerative Tools in the Control Panel. Choose the .mdb driver and associate it with an existing Access Database. The name you give in the dialog box will be the database name you use when connecting with the web.database cursor. The filename is irrelevant.
